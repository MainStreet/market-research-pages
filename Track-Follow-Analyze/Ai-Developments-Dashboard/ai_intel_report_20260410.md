### 1. Claude Mythos Withheld: First Major Model Blocked for Safety
**Policy** | Regulatory Risk | Reasoning

**Summary:** Anthropic confirmed its most powerful model (Claude Mythos) but announced it will NOT be publicly released due to cybersecurity risks — only available to select partners under "Project Glasswing." First time a frontier lab has voluntarily withheld a completed model.

**Why It Matters:** This sets a precedent that could reshape the AI release playbook. Before Mythos, the assumption was "build it, ship it." **Use Cases:** Enterprise partners accessing Mythos for advanced cybersecurity defense; Government agencies using tiered-access AI for national security; Other labs adopting similar safety evaluation frameworks before release.

**Investment Implications:** Short-term competitive disadvantage for Anthropic, but builds significant regulatory goodwill. "Tiered access" models may become industry standard. Watch for government response. Apr 9: 40+ tech/cybersecurity companies given Glasswing Preview access.

## GLOSSARY (14 terms)

**Agentic AI** — AI systems that autonomously plan, use tools, and complete multi-step tasks without human intervention at each step.

**Context Window** — Maximum text an AI can process at once. Llama 4 Maverick's 10M tokens ≈ 7.5M words (~25 novels).

**Inference** — Running a trained AI model to get outputs, as opposed to training. The primary cost driver shifting AI infrastructure economics.

**KV Cache** — Memory optimization storing previously computed attention values so models don't recalculate for every new token.

**MCP (Model Context Protocol)** — Open standard letting AI models connect to external tools through a universal interface — the "USB-C of AI."

**MoE (Mixture of Experts)** — Architecture where only a fraction of a large model's parameters activate per query, enabling massive models to run efficiently.

**Neocloud** — Cloud companies specializing exclusively in GPU-based AI infrastructure (e.g., CoreWeave), as opposed to general-purpose cloud (AWS, Azure).

**Open Weights** — AI models with publicly released parameters anyone can download, run, and modify (e.g., Gemma 4, Llama 4).

**Reasoning Distillation** — Transferring reasoning capabilities from large expensive models to smaller ones that can run on edge devices.

**Reasoning Model** — AI trained to think step-by-step before answering, using chain-of-thought techniques (e.g., OpenAI o3).

**Regulatory Sandbox** — Framework allowing companies to test AI products with relaxed regulations during development.

**SWE-Bench** — Benchmark testing AI on real-world software engineering tasks from actual GitHub issues. SWE-Bench Pro is the hardest variant.

**Vision-Language-Action Models** — AI combining vision, language understanding, and physical action into one system for robotics.

**World Models** — AI systems simulating physical environments to predict action consequences before taking them.
